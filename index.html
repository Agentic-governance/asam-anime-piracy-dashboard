📋 Methodology

Data Collection: Brave Search API (10,000+ queries across 50 languages, 150+ anime titles) producing 340K+ raw URLs from 31,454 domains.

Classification: ASAM Viewer v51 HistGradientBoosting classifier (56 features: DNS, TLS, domain name, network hop). Threshold 0.57, validated against Perplexity API ground truth (Accuracy 94.8%, Precision 95.7%, Recall 98.9%, F1 97.3%).

Catalog Extraction: Playwright headless browser with DOM extraction, network interception, pagination (up to 200 pages/site), ScraperAPI fallback for Cloudflare-protected sites. 666 sites crawled, 375K+ entries extracted.

Rights Matching: Multi-DB: MyAnimeList Jikan (30K anime) + Kitsu (22K) + Shikimori (23K, Russian↔English) + MAL multilang (24K, 20+ languages) + AniList (1.7K) + synonyms (399K expansion keys). High-precision matcher with verify_match (word overlap ≥1 OR substring ≥8 OR CJK overlap ≥2). Match rate: 43.9% (317,331/722,657), verify_rejected: 1,160. Sitemap scan ongoing — coverage will increase.

Studio Damage: Cross-referenced from original Brave crawl data (210K+ URLs across 2,234 pirate domains × 129 tracked works × 30+ languages). Each studio-work-domain combination independently verified.

OSINT Only: No accounts created. No content downloaded. All data from public sources.
