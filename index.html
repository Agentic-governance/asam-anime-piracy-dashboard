📋 Methodology

Data Collection: Brave Search API (10,000+ queries, 30 languages, 150 anime titles) + existing crawl data (210K URLs).

Classification: ASAM Viewer v51 HistGradientBoosting classifier (56 features: DNS, TLS, domain name, network hop). Threshold 0.57, validated with Perplexity API (Precision 95.7%).

Catalog Extraction: Playwright headless browser with network interception, DOM extraction, pagination, and ScraperAPI fallback for Cloudflare-protected sites.

Rights Matching: AniList GraphQL API (1,750 anime) + MyAnimeList Jikan API (193 titles) + manual curation (148 titles).

OSINT Only: No accounts created. No content downloaded. All data from public sources.
